Tab

#options

OPTION TYPE DEFAULT DESCRIPTION
default number 0 기본적으로 선택될 탭
displayClass string "is-active" 탭 컨텐츠 노출시킬 클래스 명

인스턴스 획득

                  
                  var element = document.querySelector("@selector");
                  var tab = HitaiUI.Tab.get(element);
                  // 초기화되어 있지 않다면 null을 반환합니다.
                  
                
                  
                  var tab = HitaiUI.Tab.get($("@selector"));

🟢 @selector data-autoset="tab" 이 선언되어있는 엘리먼트 입니다.

#methods

.init()

탭 초기화 합니다.

.getElement()

탭 엘리먼트를 반환합니다.

                  
                    var tab11 = $('#tab1');
                    HitaiUI.Tab.get(tab11).getElement();
                    // 해당 다이얼로그 엘리먼트 반환됨

.destroy()

탭 기능을 삭제합니다.

static

HitaiUI.Tab.get(element, initialize = false)

탭의 인스턴스를 반환합니다.

initialize 값을 true로 호출하면 초기화되지 않은 대상을 초기화 후 인스턴스를 반환합니다.

static

HitaiUI.Tab.getInstances()

현재 화면에 존재하는 탭의 모든 인스턴스를 반환합니다.

								
								var instanceList = HitaiUI.Tab.getInstances();
								// output [] or [Tab, Tab, Tab]

#events

HitaiUI.Tab.EVENT.CHANGED

탭 선택이 변경될 때 트리거 됩니다.

  • 탭2

                  document.addEventListener('UILoaded', function () {
                    $('#tab2').addEventListener(HitaiUI.Tab.EVENT.CHANGED, function(){
                      console.log('탭이 변경될때 처리')
                    })
                  });
