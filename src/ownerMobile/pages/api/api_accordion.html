Accordion

아코디언 컨텐츠를 구성합니다.

#options

OPTION TYPE DEFAULT DATA ATTRIBUTE DESCRIPTION
defaultsOpen string "" data-accordion-defaults-open 처음부터 열려있는 아이템 설정
openClass string "open" data-accordion-open-class 아코디언 오픈 시 붙일 클래스 명
toggle boolean true data-accordion-toggle 열림/닫힘 토글 여부
className.expand string "expand" 아코디언 상태 클래스 명 - 기본
className.expanding string "expanding" 아코디언 상태 클래스 명 - 변경 중
className.expanded string "expanded" 아코디언 상태 클래스 명 - 오픈 후

인스턴스 획득

								
								var element = document.querySelector("@selector");
								var accordion = HitaiUI.Accordion.get(element);
								// 초기화되어 있지 않다면 null을 반환합니다.
								
							
								
								var accordion = HitaiUI.Accordion.get($("@selector"));

@selector data-autoset="accordion" 이 선언되어있는 엘리먼트 입니다.

#selector

data-autoset="accordion"

속성이 선언된 엘리먼트는 아코디언 컴포넌트로 초기화됩니다.

#methods

.init()

아코디언을 초기화 합니다.

.open(target)

아코디언을 오픈합니다. (타겟은 aria-controls 속성이 선언 된 헤더입니다.)

								
								// 타겟은 여러가지 타입으로 받을 수 있습니다.
								.open(0); // 0번째
								.open("#acc_header1");
								.open(document.querySelector("#acc_header1"));
								.open($("#acc_header1"));

.close(target)

아코디언을 닫습니다.(타겟은 open 메서드와 동일)

.getElement()

아코디언 엘리먼트를 반환합니다.

.destroy()

아코디언을 기능을 삭제합니다.

static

HitaiUI.Accordion.redraw()

초기화 되어있는 모든 아코디언을 새로그립니다.

static

HitaiUI.Accordion.get(element)

아코디언의 인스턴스를 반환합니다.

									
										var element = document.querySelector(".accordion-wrap");
										var accordion = HitaiUI.Accordion.get(element);
										// 아코디언이 초기화되어 있지 않다면 null을 반환합니다.
									
								
									
										var accordion = HitaiUI.Accordion.get($(".accordion-wrap"));
static

HitaiUI.Accordion.getInstances()

현재 화면에 존재하는 아코디언의 모든 인스턴스를 반환합니다.

								
								var instanceList = HitaiUI.Accordion.getInstances();
								// output [] or [Accordion, Accordion, Accordion]

#events

HitaiUI.Accordion.EVENT.OPEN

아코디언이 열릴 때 트리거 됩니다.

									
										// event type string: "open.HitaiUI.accordion"
										var element = document.querySelector(".accordion-wrap");
										element.addEventListener(HitaiUI.Accordion.EVENT.OPEN, function(event) {
											var component = event.component; //component instance
											var current = event.current; //current item object
											var eventType = event.eventType; //event type
										});
									
								
									
										// event type string: "open.HitaiUI.accordion"
										$(".accordion-wrap").on(HitaiUI.Accordion.EVENT.OPEN, function(event) {
											var component = event.component; //component instance
											var current = event.current; //current item object
											var eventType = event.eventType; //event type
										});

HitaiUI.Accordion.EVENT.OPENED

아코디언이 열리고 난 후 트리거 됩니다.(애니메이션 종료 후 트리거)

HitaiUI.Accordion.EVENT.CLOSE

아코디언이 열릴 때 트리거 됩니다.

HitaiUI.Accordion.EVENT.CLOSED

아코디언이 닫히고 난 후 트리거 됩니다.(애니메이션 종료 후 트리거)
