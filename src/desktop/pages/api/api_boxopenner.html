BoxOpenner

접힘메뉴 컨텐츠를 구성합니다.

#options

OPTION TYPE DEFAULT DATA ATTRIBUTE DESCRIPTION
setupSelector function null 각 엘리먼트를 셀렉트하는 함수를 선언

인스턴스 획득

								
								var element = document.querySelector("@selector");
								var dropdown = HitaiUI.BoxOpenner.get(element);
								// 초기화되어 있지 않다면 null을 반환합니다.
								
							
								
								var dropdown = HitaiUI.BoxOpenner.get($("@selector"));

🟢 @selector data-autoset="openner" 이 선언되어있는 엘리먼트 입니다.

#selector

data-autoset="openner"

속성이 선언된 엘리먼트는 박스오프너 메뉴 컴포넌트로 초기화됩니다.

#methods

.init()

셀렉트박스를 초기화 합니다.

.updateOption(options)

박스오프너의 옵션을 업데이트합니다.

static

DropDown.updateOption(options)

초기화 되어있는 모든 박스오프너의 옵션을 업데이트합니다.

static

HitaiUI.BoxOpenner.get(element, initialize = false)

박스오프너의 인스턴스를 반환합니다.

initialize 값을 true로 호출하면 초기화되지 않은 대상을 초기화 후 인스턴스를 반환합니다.

static

HitaiUI.BoxOpenner.getInstances()

현재 화면에 존재하는 박스오프너의 모든 인스턴스를 반환합니다.

								
								var instanceList = HitaiUI.BoxOpenner.getInstances();
								// output [] or [DropDown, DropDown, DropDown]
