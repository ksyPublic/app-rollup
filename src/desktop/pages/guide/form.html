대체텍스트 : [title] / [placeholder]
  • 폼객체에 대한 대체텍스트는 title 속성을 기본원칙으로 함
  • placeholder 의 경우 스크린리더에서 음성지원되지 않는 경우가 있으므로, title 및 aria-describedby 속성으로 해당정보를 함께 제공

패스워드형(가리기/숨기기) : .input-wrap .input-wrap--password

CHECK / RADIO

  • 명시적 label : label[for] 의 경우 개발/운영상 작업공수로 인하여 title 대체(요구사항반영)
  • 암묵적 label : input[type="reaio"], input[type="checkbox"] 의 경우 UI 디자인 및 접근성을 위하여 예외적으로 암묵적 label 사용을 허용
  • 접근성 심사업체에 따라 암묵적 label을 허용하지 않을경우 공통스크립트에서 공통처리

유형1 블럭형 : .label .label--vertical
유형2 : .label--simple
유형3 : .label--md
유형4 : .label--round
유형5 : .label--btn label--{colors}
유형6 : .label--btn label--ti
